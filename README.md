> 반려동물 병원을 위한 통합 관리 시스템

**ANIDOC** 은 의료진과 보호자가 함게 반려동물의 건강을 쉽고 체계적으로 관리할 수 있도록 돕는 동물병원 전용 통합 관리 시스텝입니다. 
주요 기능으로는 진료예약 및 알림 수신, 예방접종 확인, 예약 현황 열람 등이 있습니다.
